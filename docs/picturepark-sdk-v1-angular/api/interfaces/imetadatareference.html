isRestricted

isRestricted: boolean
  • Defined in src/picturepark-sdk-v1-angular/projects/picturepark-sdk-v1-angular/src/lib/services/api-services.ts:58735

Indicates if the source of the reference is restricted because of permissions. If this is true and the SourceMetadataItemId property is filled, the user does not have the edit permission on the source metadata item. If the SourceMetadataItemId is null, the user also does not have the view permission on that item.
