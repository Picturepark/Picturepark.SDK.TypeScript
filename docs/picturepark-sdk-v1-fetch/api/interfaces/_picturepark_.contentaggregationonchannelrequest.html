Optional aggregationFilters

aggregationFilters: AggregationFilter[] | undefined
  • Defined in docs/temp/picturepark.d.ts:3881

Special filters used to filter down the aggregations' values on specific conditions. The behavior is different when filtering an aggregation that matches the same AggregationName or another aggregation. In the first case, the filter is put in "or" with (eventual) other existing filters. In the second case it is put in "and".

brokenDependenciesFilter

brokenDependenciesFilter: BrokenDependenciesFilter
  • Defined in docs/temp/picturepark.d.ts:3893

Limits the aggregation to the contents that have or not have broken references. By default it includes both.

Optional channelId

channelId: string | undefined
  • Defined in docs/temp/picturepark.d.ts:3884

Limits the simple search fields to the fields available in the specified channel. Defaults to RootChannel. For the ContentAggregationOnChannelRequest only, the existing aggregation saved on the channel are retrieved and used to perform the aggregation.
